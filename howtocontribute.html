CONTRIBUTION FLOW

How To Contribute

Follow these simple steps to contribute to UIverse.

01

Fork Repository

Fork the official UIverse repository to your GitHub account.

02

Clone Project

Clone your forked repository locally using Git.

03

Create Component

Build modern responsive UI components using HTML, CSS and JavaScript.

04

Push Changes

Commit your code and push the changes to your repository.

05

Create Pull Request

Submit your pull request for review and contribution.

06

Get Merged

Once approved, your component becomes part of UIverse.

GUIDELINES

Contribution Guidelines

Use clean and reusable code
Keep components responsive
Follow modern UI design principles
Avoid unnecessary dependencies
Use proper folder structure
