Frequently Asked Questions

How does URL state persistence work?

Application state is stored inside URL parameters and restored automatically when the page loads.

Can I share filtered views?

Yes. Generated URLs include all current filters, search queries and sorting options.

Does it support browser history?

Fully compatible with pushState, replaceState and back-forward navigation.
