# Global CSS Variables and Design Tokens

This document details layout tokens, typography scale, radii, and theme states used across components.

## Theme Colors
- `--color-primary`: Main brand accent color.
- `--color-secondary`: Subdued background panel color.
- `--bg-panel`: Background panel color.

## Spacing & Grid System
- Use standard margins: `8px`, `16px`, `24px`, `32px`.
