Documentation

Welcome to UIverse documentation. UIverse is an open-source library of reusable UI components crafted with pure HTML, CSS, and JavaScript.

Quick Start Guide

  1. Explore our collection of components from the sidebar navigation.
  2. Click on a component card to preview it and copy its HTML/CSS code.
  3. Paste the code directly into your project - no frameworks or external configuration required.
